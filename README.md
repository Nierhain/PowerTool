# PowerTool

Ever asked yourself "What if the harvesting tool from Satisfactory was available in Minecraft?"?
Now  you have the answer!

## Features

- Pickaxe, Shovel and Axe in one
- uniform break speed
- AoE mining
- upgradable with magnet, fortune and more
